## Commands
- `npm run compile` — Compile Solidity + ABI export (`data/abi/`) + contract sizer + TypeChain (`typechain/`)
- `npm test` — Run mocha/chai tests in `test/`
- `REPORT_GAS=1 npm test` — Include gas report
- `npm run clean` — Remove `cache/` and `artifacts/`
- `npm run node` — Start local Hardhat node
- `npm run run:sepolia <script>` — Run a Hardhat script on Sepolia
- `npx hardhat deploy --network <network>` — Deploy via `deploy/` scripts (hardhat-deploy)

## Architecture
- `contracts/` — Solidity sources
- `test/` — Mocha/Chai unit tests (import `ethers` from `"hardhat"`, types from `"../typechain"`)
- `deploy/` — hardhat-deploy deployment scripts (use `DeployFunction` pattern with `namedAccounts`)
- `deployments/` — Per-network artifacts (auto-generated by hardhat-deploy)
- `scripts/` — Hardhat `run` scripts (use `hre.ethers`)
- `etherTest/` — Standalone ethers.js scripts (no Hardhat; uses `dotenv/config` + `ethers` directly)
- `typechain/` — Auto-generated TypeScript bindings (gitignored; must compile first)
- `data/abi/` — Auto-generated ABI JSON (gitignored)

## Setup
1. Copy `.env_example` to `.env`
2. Required vars: `TESTNET_PRIVATE_KEY` (deploy), `ETHERSCAN_API` (verify)
3. `REPORT_GAS=1` enables gas reporter
4. `.env` is gitignored
